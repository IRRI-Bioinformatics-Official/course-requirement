Working on Remote Server

WINDOWS Users


Recommended Tools

  • Remote Desktop Connection (RDP) – for Windows servers
  • PowerShell / Command Prompt – for SSH access
  • PuTTY (if provided)

Connecting

  1. Open Remote Desktop Connection or PowerShell.
  2. Enter the server address (IP or hostname).
  3. Log in with your assigned username and password or SSH key.
  4. If required, connect to the VPN first.

Working on the Server

  • Verify you are logged in to the correct system.
  • Work only in assigned folders.
  • Follow file and naming conventions.

Best Practices

  • Do not install software unless approved.
  • Avoid running unknown or destructive commands.
  • Save work frequently.

Logging Out

  • Close applications.
  • Sign out of the remote session (do not just close the window).
